# EasyTamilTools 🇮🇳

EasyTamilTools is a free, open-source web application that provides **Tamil & English letter formats, calculators, and daily-use online tools** built using **Python Flask**.

🌐 Designed specially for Tamil users  
⚡ Simple, fast & SEO-friendly  
🆓 100% Free & Open Source  

---

## ✨ Features

### 📝 Letter Formats
- School Leave Letter (Tamil & English)
- Sick Leave Letter
- Office Leave Letter
- Bank Account Closing Letter
- Aadhaar Correction Letter
- Editable, Copy & Print supported

### 🧮 Calculators
- Age Calculator
- BMI Calculator
- EMI Calculator
- Percentage Calculator

### 🛠 Tools
- Text Counter
- Date Difference Calculator
- Case Converter
- Number to Words
- Unit Converter
- Image to Text (Tamil + English OCR)

---

## 🧑‍💻 Tech Stack
- Python 3
- Flask
- HTML5, CSS3
- Jinja2 Templates
- Tesseract OCR
